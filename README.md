# 常用依赖

目录结构

~~~
+-- src
|   +-- base64.js
|   +-- bindCustomEvent.js
|   +-- dispatch.js
|   +-- templater.js
+-- index.js
~~~
模块介绍

+ base64，使用base64编码和解码字符串

+ dispatch，事件代理用于事件分发
+ templater，模版引擎
